# Sales Forecasting & Demand Intelligence System

## Overview
A machine learning-based dashboard for sales forecasting, anomaly detection, and product demand segmentation.

## Features
- Sales trend analysis
- ARIMA Forecasting
- SARIMA Forecasting
- Prophet Forecasting
- XGBoost Forecasting
- Isolation Forest Anomaly Detection
- Z-Score Anomaly Detection
- Product Demand Segmentation using K-Means
- KPI Dashboard

## Tech Stack
- Python
- Pandas
- Scikit-Learn
- XGBoost
- Prophet
- Streamlit
- Matplotlib
- Seaborn

streamlit run app.py
